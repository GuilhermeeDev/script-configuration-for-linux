# script-configuration-for-linux
Script de configuração para distros linux e baseadas em linux.

- Acessar o dir raiz do script:
```
cd src
```

- Dar permissão para execução do script:
```
chmod +x ./main.sh
```

- Executar o script: 
```
./main.sh
```
